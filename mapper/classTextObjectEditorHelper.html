Detailed Description

Helper class to enable text editing (using the DrawTextTool and the EditTool).

To use it, after constructing an instance for the TextObject to edit, you must pass through all mouse and key events to it and call draw() & includeDirtyRect(). If mousePressEvent() or mouseReleaseEvent() returns false, editing is finished.
