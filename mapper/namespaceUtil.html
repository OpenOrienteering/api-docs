Used to implement arbitrarily rotated grids which are constrained to an axis-aligned bounding box.

This functions calls processor.processLine(QPointF a, QPointF b) for every line which is calculated to be in the given box.

Parameters
extentExtent of the box.
horz_spacingHorizontal spacing of the lines.
vert_spacingVertical spacing of the lines.
horz_offsetHorizontal offset of the first line from the origin.
vert_offsetVertical offset of the first line from the origin.
rotationAngle used to rotate the lines.
processorCallback object on which processor.processLine(QPointF a, QPointF b) will be called for each line.
