Detailed Description

Map undo step which adds the contained objects.

Take care of correct application order when mixing with a delete step to make sure that the object indices are preserved.

void AddObjectsUndoStep::removeContainedObjects ( bool  emit_selection_changed)

Removes all contained objects from the map.

This can be useful after constructing the undo step, as it is often impractical to remove the objects directly when adding them as the indexing would be changed this way.
