bool GPSDisplay::checkGPSEnabled ( )

Checks if GPS is enabled and may guide the user to the device settings.

Checks if GPS is enabled in the device settings. If this is not the case, it asks the user whether he wishes to open the device's location settings dialog.

Returns true if GPS is enabled. Must return true also if the settings dialog remains open when returning from this function (i.e. the final result is not known. May return false if GPS remains disabled.
