Detailed Description

Transformation parameters for non-georeferenced templates.

The parameters are to applied in the order

  1. translate
  2. rotate
  3. scale.

So this order is also chosen for the member variables, and thus used in list initalization.

See also
Template::applyTemplateTransform()
