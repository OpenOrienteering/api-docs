Detailed Description

Shows all symbols from a map in a scrollable widget.

SymbolWidget shows all symbols from a map's symbol set. It lets the user select symbols and perform actions on symbols.

The implementation is based on SymbolRenderWidget and QScrollArea. Normally it is used inside a dock widget.
