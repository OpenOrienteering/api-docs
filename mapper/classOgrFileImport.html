Detailed Description

An Importer for geospatial vector data supported by OGR.

OGR needs to know the filename. The filename can be either derived from a QFile passed as stream to the constructor, or set directly through the option "filename".

The option "separate_layers" will cause OGR layers to be imported as distinct map parts if set to true.
