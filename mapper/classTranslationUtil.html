QString TranslationUtil::localeNameForFile ( const QString &  filename)
static

Returns the locale name for a given translation file, or returns an empty string if the file is not a valid translation.

In order to be valid, the file must exist, and the file name (without directory) must be composed of the base name, an underscore, the locale name and the suffix ".qm".
